# Squiggle-lite

Squiggle Lite is a **simplified web version** of **Squiggle**, an ADHD-friendly focus and mood companian currently being developed as a mobile app.

This project exists to explore core product ideas - low-friction check-ins, gentle accountablitiy, and visual pattern awareness - in a fast, iteration-friendly format.
